    电子墨水屏¶

    2in9记录¶

    从汉朔2.9寸电子价签中拆出的墨水屏可以使用微雪电子的V1版本程序进行驱动，有几点需要记录一下

    1. 墨水屏较长的一侧为驱动芯片的高侧，较短的一侧为驱动芯片的宽侧，示意图如下
           0               128
        ----------------        01234567 ... 128
0      |                |       ******** ...  * 
       |                |       在宽侧每8个像素点为一组，字节倒序，即pixel 0为MSB，
       |                |       pixel 7为LSB，取模时需要注意
       |                |
       |                |
       |                |
       |                |
       |                |
       |                |
       |                |
       |                |
       |                |
       |                |
       |                |
       |                |
296    |                |
        ----------------
    1. 电子价签内部有一个MOS-FET用于控制墨水屏的供电，链接在P2.6，该pin默认为高电平，可以使墨水屏正常供电，可以忽略不管。
